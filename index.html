MockOver

MockOver is a simple bookmarklet that allows you to specify an image and have it overlay your developed web page. You can then drag it around or use keyboard shortcuts to fine tune where it is positioned and how semitransparent it is.

Drag this to your bookmarks toolbar: MockOver

The unpacked source is publicly available.

Keyboard Shortcuts

Up (↑)
Moves image up 1px
Down (↓)
Moves image down 1px
Left (←)
Moves image left 1px
Right (→)
Moves image right 1px
Plus (+)
Increases opacity by 0.05
Minus (-)
Decreases opacity by 0.05

Thanks to Jason Siffring for the inspiration.

Notes

Currently, this will only work with local developments (i.e. you can't preview a local mockup against an external HTML file). Working on making this possible via HTML5 soon.
